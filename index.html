Light the Y

Welcome to the BYU STEM Camp "Light the Y" activity! We're happy to have you. Once you have your Y-badge PCB board, you're ready to get programming.

These online modules are designed to be used with an instructor or as a self-paced program. Feel free to take as much time as needed to complete each module. If you already feel comfortable with one of the topics discussed in these activities, feel free to challenge yourself and think outside the box. The Y-badge code GitHub repository contains all the code needed to use the modules on this website. For instructions on cloning this GitHub repo to your computer, consult the Software Setup tab from the left menu.

Happy coding!
